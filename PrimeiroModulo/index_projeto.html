Estrutura e organizacao do conteudo

(Voltar)

As tags h1 até h6 são usadas para definir títulos, indo do mais importante (h1) ao menos importante (h6). Elas ajudam tanto na organização visual quanto na estrutura semântica de uma página, essencial para SEO e acessibilidade. A tag p define um parágrafo, sendo usada para agrupar blocos de texto. Já a hr insere uma linha horizontal que separa visualmente seções de conteúdo, funcionando como um divisor temático. A tag blockquote é usada para destacar citações longas de fontes externas, geralmente com indentação.

Estilizacao e enfase de texto

(Voltar)

Este bloco inclui tags que alteram a aparência ou o significado do texto. A strong aplica negrito com ênfase semântica (dizendo que aquele conteúdo é importante), enquanto i aplica itálico, geralmente para nomes de obras ou palavras em outro idioma. A u sublinha o texto, e a mark destaca visualmente como um marca-texto. A small exibe o texto em um tamanho menor, geralmente usado para rodapés ou informações adicionais. As tags sub e sup exibem texto em formato subscrito e sobrescrito, como em fórmulas químicas (H₂O) ou matemáticas (x²). A del indica que um texto foi removido (riscado), e a abbr mostra o significado de uma abreviação quando o usuário passa o mouse por cima.

Listas, links e navegacao

(Voltar)

Neste grupo, temos elementos usados para criar listas e navegação. A ol define uma lista ordenada, onde os itens aparecem numerados, enquanto a ul define uma lista não ordenada, com marcadores simples. Dentro de ambas, a tag li define cada item da lista. A a é usada para criar hiperligações (links), podendo levar o usuário para outra página, um arquivo, ou mesmo outra parte da mesma página. Por fim, a tag font, embora obsoleta, era utilizada para definir diretamente cor, tamanho e tipo da fonte — hoje em dia, essas estilizações são feitas com CSS e a tag font deve ser evitada.

Estrutura Basica de um HTML

(Voltar)

A estrutura básica de um documento HTML é composta por três partes principais: a tag html, o head e o body. Tudo o que será exibido ou configurado em uma página precisa estar dentro da tag , que é a raiz do documento. Dentro dela, o head funciona como o cabeçalho invisível da página, sendo o local onde colocamos informações como o título da aba do navegador (usando a tag title), a codificação de caracteres (como meta charset="UTF-8"), além de links para arquivos de estilo CSS, fontes e scripts JavaScript. Já o body é a parte visível da página, onde ficam os elementos que o usuário vê e interage, como textos, títulos, imagens, links, listas e formulários. Antes dessas três partes, colocamos a declaração !DOCTYPE html no topo do arquivo, que não é uma tag HTML, mas sim uma instrução para o navegador entender que se trata de um documento HTML5. Essa estrutura é essencial e deve estar presente em qualquer página HTML, mesmo nas mais simples.
